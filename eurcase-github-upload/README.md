# Eurcase Website

Static first-stage website for Eurcase.

## Files

- `index.html` - homepage
- `src/styles.css` - visual styles
- `PROJECT_CONTEXT.md` - project decisions, naming rules, and current working context
- `CHAT_HANDOFF_2026-06-17.md` - conversation handoff summary for continuing the project with another Codex account
- `IMAGE_MAP.md` - homepage and site image naming map
- `_headers` - Cloudflare Pages security headers
- `robots.txt` - crawler rules
- `sitemap.xml` - basic sitemap

## Cloudflare Pages Settings

- Framework preset: None
- Build command: leave empty
- Build output directory: `/`

## Domain

Production domain:

- `eurcase.com`
- `www.eurcase.com`
